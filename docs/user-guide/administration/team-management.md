# Team Management

Manage your team members and control access to your TrueKredit workspace.

---

## Team Member Limits

Each tenant can have a **maximum of 5 team members**, including the owner.

| Role | Count Towards Limit |
|------|---------------------|
| Owner | Yes |
| Admin | Yes |
| Staff | Yes |

If you need additional team members, please contact support.

---

## User Roles

TrueKredit uses three roles to control access:

### Owner

- Full access to all features
- Can manage billing and subscription
- Can add, edit, and remove team members
- Can delete the tenant account
- Cannot be removed or demoted

### Admin

- Can manage loans, borrowers, and products
- Can add and edit team members (except Owner)
- Can view billing information
- Can access reports and exports

### Staff

- Can create and manage borrowers
- Can create loan applications
- Can view loans and products
- Limited access to settings

---

## Adding Team Members

1. Navigate to **Settings** from the sidebar
2. Scroll to the **Team Members** section
3. Click **Add User**
4. Fill in the required information:
   - **Email**: The user's email address
   - **Name**: (Optional) Display name
   - **Password**: Temporary password for new users
   - **Role**: Select Admin or Staff
5. Click **Add User** to send the invitation

The new team member can immediately log in using the provided credentials.

---

## Managing Existing Members

### Activating/Deactivating Users

Owners can deactivate team members without removing them:

1. Go to **Settings** > **Team Members**
2. Find the user in the table
3. Click the activate/deactivate button

Deactivated users cannot log in but their data and audit history is preserved.

### Changing Roles

Currently, role changes require removing and re-adding the user with the new role. Contact support if you need to change a user's role.

---

## Multi-Tenant Access

A single user email can belong to multiple tenants with different roles. For example:

- `john@example.com` can be an **Owner** at "Company A"
- `john@example.com` can be a **Staff** at "Company B"

Use the **tenant switcher** in the sidebar to switch between tenants.

---

## Security Best Practices

- Use strong, unique passwords for each team member
- Regularly review team member access
- Deactivate users who no longer need access
- Monitor the login history in Settings > Security
